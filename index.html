lorg

A framework and javascript free privacy respecting Google Text/Image Proxy originally based on a cut-down version of LibreX but now evolved into simple, lightweight API driven Meta Search Engine.

Check out the Demo Search Engine

Pronounced Lur-Ugh - An Irish word meaning "Track, trace; seek, search for."

Features

  • Ad free
  • JavaScript free
  • Tracking snippets from URLs are removed
  • Easy to use JSON API for developers
  • No 3rd party libs / frameworks are used
  • Themable
  • Customizable
  • Use Google for Text and Image search.
  • Uses Invidious for Youtube Search

Special Queries

  • Currency Convertor (type: '12 GBP to EUR')
  • Current weather (search for 'weather')
  • Your External IP (type: 'my ip')
  • Your User Agent (type: 'my ua')
  • Word definition(type: 'run mean')
  • Automatically search Wikipedia (type: 'star wars')

Easy to host

lorg can be deployed as a Docker container or as a Standalone installation.

System Requirements:

  • nginx
  • PHP v7 or higher
  • PHP Modules:
    • DOM
    • Curl
